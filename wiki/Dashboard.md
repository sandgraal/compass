# Dashboard

**Route:** `/dashboard` · **Sidebar:** Dashboard · **⌘K:** "Dashboard"

The Dashboard is your landing page and morning brief — a single glance at what matters today. It
is the default view after launch and after onboarding.

## What it shows

- **Today's Tasks** — the checklist items scheduled for the current day, pulled from the same
  store as the [Daily](Planner-Daily-Weekly-Monthly#daily) view. Tasks come from manual entry and
  from your integrations (calendar, GitHub, Gmail).
- A condensed view of the day's signals so you can decide where to go next (Daily for the full
  list, Finance for money, Knowledge for notes).

## Working from the Dashboard

- Check off tasks directly as you complete them.
- Use the **[Command Palette](Search-and-Command-Palette)** (⌘K) to jump anywhere or to add a
  "New task for today" without leaving the page.

> The Dashboard is intentionally lightweight — it's a *brief*, not a workspace. The deep daily
> workflow (calendar strip, GitHub due-today, Gmail action items, habits, templates) lives on the
> **[Daily](Planner-Daily-Weekly-Monthly#daily)** page.
