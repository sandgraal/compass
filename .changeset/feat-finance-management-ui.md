Finance page now supports full account, transaction, and rules management. The page is split into Overview / Transactions / Accounts / Rules tabs. Add, edit, and delete accounts (checking, savings, credit, investment) including APR, minimum payment, credit limit, and a "treat as debt" toggle. Click any transaction row to expand an inline editor for category (predefined list + custom), subcategory, account, and notes; hover-reveal a delete affordance. A new Rules tab manages categorization patterns sorted by priority with an info note clarifying that rules apply only to newly-ingested transactions.
